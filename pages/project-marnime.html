MaRnime 🌌

Website anime modern dengan tema Aurora Dark yang elegan. Jelajahi, lacak, dan kelola daftar anime kamu menggunakan data dari AniList GraphQL API. Dilengkapi watchlist, profil, dan multibahasa.

Next.js 16ReactTypeScriptTailwind CSS 4Framer MotionSupabaseAniList GraphQLVercel
Live Demo Source Code

✨ Features

✦
Jelajahi Anime — Filter berdasarkan genre, format, status, musim, dan tahun
✦
Detail Anime — Sinopsis, karakter, staf, relasi, rekomendasi, dan trailer
✦
Anime Musiman — Anime sedang tayang per musim dengan countdown episode
✦
Watchlist — Tambah anime ke daftar tontonan (Planning, Watching, Completed)
✦
Autentikasi — Daftar dan masuk dengan email & password via Supabase
✦
Multibahasa — Toggle bahasa English / Indonesia

🛠 Tech Stack

FrontendNext.js 16, React, TypeScript
StylingTailwind CSS 4, Custom Aurora Dark Theme
AnimationFramer Motion
Data APIAniList GraphQL API
Auth & DBSupabase (Auth + PostgreSQL)
HostingVercel
