MaRticle Design ✨

Interactive particle system designer premium yang dibangun dengan vanilla HTML, CSS, dan JavaScript. Buat efek partikel stunning secara real-time dengan visual editor intuitif, multi-layer system, dan 18 preset bawaan.

HTML5CSS3JavaScript Canvas 2D APIlocalStorageVercel
Live Demo Source Code

✨ Features

✦
18 Built-in Presets — Fire, Rain, Snow, Smoke, Sparkle, Explosion, Confetti, Galaxy, Aurora, dan lainnya
✦
Multi-Layer System — Jalankan beberapa efek partikel secara bersamaan di satu canvas
✦
Mouse Interaction — Klik dan drag di canvas untuk emit partikel di posisi kursor
✦
Export Options — Export ke JSON config, WebM recording, PNG frame, dan copyable JS code
✦
Undo/Redo — Full history dengan Ctrl+Z / Ctrl+Y support
✦
Shareable Links — Encode konfigurasi ke URL dan bagikan ke orang lain

🛠 Tech Stack

CoreHTML5, CSS3, JavaScript (ES6+)
RenderingCanvas 2D API
StylingVanilla CSS, Glassmorphism, Dark Theme
TypographyGoogle Fonts (Inter)
StoragelocalStorage (custom presets)
HostingVercel
