GrabMaR 💰

Aplikasi kalkulator pendapatan harian untuk driver Grab. Menghitung uang digital dan cash secara terpisah, menyimpan riwayat pendapatan dengan login akun, dan melihat detail breakdown setiap hari kerja.

HTMLCSSJavaScriptTailwind CSSNode.jsSupabaseJWTVercel
Live Demo Source Code

✨ Features

✦
Kalkulator Pendapatan — Input pendapatan dari GrabFood, GrabCar, GrabBike dengan mudah
✦
Live Total — Total pendapatan update real-time saat menginput
✦
User Accounts — Login dan register untuk menyimpan data secara personal
✦
History Saving — Simpan riwayat pendapatan harian ke database
✦
Detail Breakdown — Lihat detail breakdown per kategori dan hari
✦
Responsive — Tampilan optimal di smartphone untuk driver

🛠 Tech Stack

FrontendHTML, CSS, JavaScript, Tailwind CSS
BackendNode.js Serverless Functions
DatabaseSupabase (PostgreSQL)
AuthJWT (JSON Web Token)
HostingVercel
