MaRental 🚗

Aplikasi rental mobil online full-stack. Fitur booking otomatis, pilih driver, upload bukti pembayaran via Cloudinary, riwayat rental, admin dashboard lengkap, dan auto-release kendaraan.

BladeTailwind CSSLaravel 10PHP 8.2SupabaseCloudinaryRender
Live Demo Source Code

✨ Features

✦
Katalog Kendaraan — Lihat daftar mobil yang tersedia dengan spesifikasi dan harga
✦
Booking Otomatis — Pemesanan otomatis dengan kalkulasi durasi dan total harga
✦
Pilih Driver — Opsi lepas kunci atau dengan driver yang tersedia
✦
Upload Bukti Bayar — Upload bukti pembayaran via Cloudinary
✦
Admin Dashboard — Kelola kendaraan, pesanan, driver, dan pembayaran
✦
Auto-Release — Kendaraan otomatis tersedia kembali setelah masa rental selesai

🛠 Tech Stack

FrontendBlade Templating, Tailwind CSS, Vite
BackendLaravel 10, PHP 8.2
DatabaseSupabase (PostgreSQL)
StorageCloudinary (Image Upload)
HostingRender
