SafeNurse 🏥

Sistem pelaporan insiden keselamatan pasien berbasis web untuk rumah sakit. Perawat melaporkan insiden melalui asisten chatbot AI, dan laporan diproses secara otomatis dengan validasi, scoring, serta notifikasi ke pihak terkait.

Live Demo Source Code

✨ Features

✦
Chat Assistant AI — Pelaporan insiden melalui chatbot interaktif dengan AI yang membimbing pengisian data
✦
Voice Input — Input suara menggunakan Web Speech API langsung dari browser
✦
Auto Kategorisasi — Data laporan dikategorikan otomatis oleh LLM (KTD, KPC, KNC, KTC, Sentinel)
✦
Multi-Role Dashboard — Dashboard berbeda untuk Perawat, Kepala Ruangan, Chief Nursing, dan Super Admin
✦
Notifikasi & Email — Notifikasi real-time dan email otomatis untuk setiap perubahan status
✦
Scoring & Grading — Penilaian risiko insiden berdasarkan dampak dan probabilitas

🛠 Tech Stack

FrontendNext.js, React, TypeScript, Tailwind CSS
BackendNode.js, Express.js
DatabaseSupabase (PostgreSQL)
AI/LLMOpenRouter (multi-model fallback)
EmailNodemailer (Gmail SMTP)
HostingVercel
