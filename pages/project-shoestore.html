R&R ShoeStore 👟

Aplikasi e-commerce sepatu full-stack dengan sistem rekomendasi berbasis machine learning NMF. Dibangun dengan React dan Flask, dilengkapi panel admin lengkap untuk mengelola produk, pesanan, dan data pengguna.

ReactTailwind CSSFlaskGunicornSupabasescikit-learn (NMF)JWT
Live Demo Source Code

✨ Features

✦
Katalog Produk — Jelajahi koleksi sepatu dengan filter kategori, pencarian, dan pagination
✦
Rekomendasi AI — Sistem rekomendasi personal menggunakan NMF machine learning
✦
Keranjang & Checkout — Tambah ke keranjang, atur jumlah, dan proses pesanan
✦
Multi-Payment — Transfer Bank, E-Wallet, COD, Kartu Kredit
✦
Wishlist — Simpan sepatu favorit untuk dibeli nanti
✦
Panel Admin — CRUD lengkap untuk kategori, produk, user, pesanan, dan interaksi

🛠 Tech Stack

FrontendReact, Tailwind CSS, Axios
BackendFlask, Flask-JWT-Extended, Gunicorn
DatabaseSupabase (PostgreSQL)
ML Modelscikit-learn (NMF), Pandas, NumPy
HostingRender
